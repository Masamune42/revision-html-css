Media Queries

Permet de définir des règles différentes de style en fonction de la taille d'écran

Combinaison des règles à l'aide de : only, add et not
  • color
  • min-height
  • min-width
  • min-device-height
  • min-device-width
  • min-device-orientation : portrait ou paysage
  • media :
    • screen : écran classique
    • handled : périphérique mobile
    • print : impression
    • tv : télévision
    • projection : projecteur
    • all : tous les type d'écran
